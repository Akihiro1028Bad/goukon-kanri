**代替案1: 完全な README.md を作成する**
- 理由: Issue #60 の要件は「HTML コメントを1行追加」のみであり、完全な README の作成はスコープ外

**代替案3: 複数行のコメントや追加情報を含める**
- 理由: Issue の要件は「1行追加」と明示されており、スコープを逸脱する
